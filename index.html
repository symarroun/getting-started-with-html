Thinking Lenses

As a programmer, you have to develop specific thinking lenses that you will alternate between as you are creating your code. Each len has its advantages for each stage of the coding process. The lenses are as follow:

1. Procedural Thinking


Like saying, (1) ..., (2) ..., (3) ..., and so on.

2. Abstract Thinking


Like saying, 1 + 1 = 2

3. Systematic Debugging


Like saying, The problem is ..., and the possible causes coule be ...

4. Technological Empathy


This is the lens where the user get to understand how the computer sees things.

5. Systems Thinking


This is the most important lens. This lens helps breaking down big ideas into component pieces.

Terms

HTML Element:
Elements types are indicated by HTML tags. Tags are written like this < > and a letter in between indicating what tag it is. Each element has an opening tag and a closing tag; Except some element like the ones for an image . The element without closing tags are referred to as an element with a void tag.
Elements are divided in two main categories: Inline and Block.
  • Inline Elements: a, span, br, img, strong
  • Block elements: div, p, form

Web Page:
A web page is a text document written in a language called HTML. Web browsers read these documents, and then interpret and display them.
World Wide Web (WWW)
The World Wide Web (WWW) is a collection of HTML documents which are connected through this web.

Tags

Keep in mind that if a tag is not closed, everything after the tag is activated by that tag. Yes, computers are stupid. They do exactly what YOU tell them to do. Thus, be careful about how you communicate with a computer and understand how it understands you :).

  • <i> & <em> : Both make the content italics. However, emphasis <em> is becoming more pupolar for advanced coding reasons
  • <b> & <strong> : bold vs. strong, they both bold the word.

HTML Elements Attributes

Some elements requires attribues to be added within a tag. For example, an images needs to be directed to its source and a link (URL) should be referenced to its URL link. The attribute is included in the opening tag in this form:
< a href="URL" > Content . In other words, an element with an attribute with have the following components: Within the opening tag (an attribute name, a value), and content followed by a closing tag where appropriate.
